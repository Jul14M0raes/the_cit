# Site institucional do CIT

O objetivo do projeto é criar um site que sirva de vitrine institucional idependente para o CIT (Centro de Inovação e Tecnologia) Porto Velho Zona Norte

## Escopo do projeto (Páginas)
### Público
- Galeria: galeria de fotos de eventos antigos, membros antigos, memorial 

- artigos do cit: breve visualização de trabalhos escritos pelo CIT

- Pesquisa: Projetos em desenvolvimento

- Eventos: eventos do cit, workshops e etc..

- Equipe: mostrar a equipe atual do cit
